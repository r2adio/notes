# GIT LOG:

- `git log` shows a history of commits in a repo.
- this helps u see:
  - who made a commit
  - when the commit was made
  - what was changed

## Commit Hash:

- each commit has a unique identifier, **commit hash**
- commit hash: long string of alpha numeric characters

## Common log cmds used:

- `git log`
- `git --no-pager log`
- `git --no-pager log -n 10`
- `git --no-pager log -n 10 --oneline`
- `git --no-pager log -n 10 --oneline --parents`
- `git --no-pager log -n 10 --oneline --parents --graph`
